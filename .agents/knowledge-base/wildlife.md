# Wildlife Photography Expert Workflow (2026)

## 1. Core Module Sequence
For wildlife (birds/mammals), the signal must be cleaned before acutance is applied.
1. **Denoise (profiled)**: Use Wavelets in Y0U0V0 mode. Prioritize Chroma denoise.
2. **Capture Sharpen**: Enable in the Demosaic module (8-10 iterations).
3. **AgX**: Set Pivot Relative Exposure directly on the subject to maximize relevant contrast.

## 2. Surgical Enhancement
- **Diffuse or Sharpen**: Apply only to the subject (e.g., eyes, fur) using drawn masks.
- **Preset**: "lens deblur | medium" for slightly soft shots.

## 3. Composition
- **Lead Room**: Ensure the crop provides space in the direction of the animal's gaze.
- **Cinematic**: Consider a 16:9 or 2.35:1 aspect ratio for animals in wide environments.
